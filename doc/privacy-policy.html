This privacy policy describes how the Skip Wait Chrome extension (“Extension”) handles information. The Extension bypasses countdown timers and skip-wait pages on supported link shorteners and file hosts.

  1. 1. Permissions

    The Extension requests the following Chrome permissions:

    • Scripting — Used to run bypass logic on pages you visit.

  2. 2. Data handling

    The Extension does not collect, store, or transmit your personal data. It does not use Chrome storage, and it does not use analytics, tracking, or telemetry.

    On certain supported sites, the Extension may read or clear data stored by that site in your browser (for example, in that page’s localStorage) only to perform the bypass. That data is not sent to the developer or any third party.

  3. 3. Third parties

    The Extension does not send any data to the developer or to any third-party services. Any network requests made by the Extension are only to the current page or to the link shortener or file host you are using, as required to perform the bypass.
