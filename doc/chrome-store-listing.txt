Skip Wait bypasses countdown timers and waiting pages on link shorteners, safelinks, and file hosts so you get the final URL or download with no delay. Skip countdown and bypass wait timers on short links and file hosts without an account, setup, or paywall. The extension runs only on supported pages; the rest of your browsing is unchanged.


--------------------------------------------------------------------------------
OVERVIEW
--------------------------------------------------------------------------------

Skip Wait removes mandatory wait times on supported sites. You either get an instant redirect to the destination or a one-click way to skip the timer. It works on short links with wait timers, safelink and redirect pages, file hosts that use countdowns, and pages that unlock after a delay. Unsupported pages are never modified.


--------------------------------------------------------------------------------
SUPPORTED SITES & BEHAVIOR
--------------------------------------------------------------------------------

• Link shorteners and safelinks - direct redirect to the final URL when possible
• File hosts and download pages - one-click skip so you can start the download
• Streaming and unlock pages - bypass countdown or cookie/session delay
• New sites are added with updates; use the popup to request support for more


--------------------------------------------------------------------------------
FEATURES
--------------------------------------------------------------------------------

• Runs automatically on supported pages when enabled - optional master switch in the popup turns all automation off or on globally
• Zero site-by-site setup - open the link as usual; the extension handles the rest when it is on
• Free - no subscription, sign-up, or paywall
• Lightweight - no impact on sites that are not supported wait or short-link pages
• Request new sites via the popup for future releases


--------------------------------------------------------------------------------
HOW TO USE
--------------------------------------------------------------------------------

Install the extension and open any supported short link or wait page. You get an instant redirect or an on-page button to proceed. No setup or account required.


--------------------------------------------------------------------------------
SINGLE PURPOSE
--------------------------------------------------------------------------------

Skip Wait has one purpose: to bypass countdown timers and waiting pages on link shorteners and file hosts so you get instant access to the final URL or download. It does not collect data, show ads, modify unrelated pages, or bundle unrelated features. It runs only on supported wait pages and short links. One extension, one job.


--------------------------------------------------------------------------------
SCRIPTING PERMISSION
--------------------------------------------------------------------------------

Skip Wait requests the "scripting" permission to run code on wait pages and short-link pages. It needs to run in the page context to detect countdown timers, resolve redirect chains, find or click skip/unlock controls, and inject controls when the host page does not provide them. Content scripts alone cannot reliably bypass many wait-page implementations. Scripting is used only on supported wait and short-link pages, and only for bypassing countdowns and waiting pages - not for analytics, ads, or any other purpose.


--------------------------------------------------------------------------------
STORAGE PERMISSION
--------------------------------------------------------------------------------

Skip Wait requests the "storage" permission (chrome.storage.local) to save your global on/off choice from the extension popup and short-lived bypass progress for multi-step shortener flows (for example, v2links / Cut4Money-style chains on shr2.link and nitro-link.com). That data stays on your device, is not sent to us or any third party, and is not used for analytics or ads. The popup, background service worker, and content scripts share the same settings so bypass behavior stays consistent until you change it or the flow completes. No personal data, browsing history, or full page content is written to storage.


--------------------------------------------------------------------------------
COOKIES PERMISSION
--------------------------------------------------------------------------------

Skip Wait requests the "cookies" permission so the background service worker can read a site-set cookie on certain prolink shortener pages when the final destination URL is stored only in an HttpOnly cookie. Content scripts cannot access HttpOnly cookies, so without this permission the extension cannot finish the bypass on those flows. On a supported prolink page, the content script asks the background worker to read cookies for that tab's origin only, decode the embedded destination URL on your device, and redirect you. Cookie values are not uploaded, shared with third parties, or used for analytics or ads. The extension does not set, change, or delete cookies—only reads them to resolve the redirect on supported wait pages.


--------------------------------------------------------------------------------
WEB NAVIGATION PERMISSION
--------------------------------------------------------------------------------

Skip Wait requests the "webNavigation" permission for a narrow use on certain v2links shorteners (including shr2.link and nitro-link.com). Those sites redirect away almost instantly, before a page script can run. The background service worker listens for navigation to those short-link URLs only, fetches the first redirect target from the shortener, and updates the tab so you reach the verification article without losing the bypass. It is not used to track where you browse in general—only to intercept supported shortener entry URLs in time. The final download URL always comes from the shortener’s own API response, not from a hardcoded destination.


--------------------------------------------------------------------------------
HOST PERMISSION
--------------------------------------------------------------------------------

Skip Wait requests host permission to run on the same origins as the wait pages and link shorteners it supports. Bypassing a countdown or short link requires running code on that page; without host access it cannot read the page, resolve redirects, or perform the required actions. Supported sites include many link shorteners, file hosts, and safelink domains; the set grows with updates. Host permission is limited to what is needed to operate on those pages. It is not used to collect data or alter unrelated sites - only to skip wait timers and countdowns on supported pages.
